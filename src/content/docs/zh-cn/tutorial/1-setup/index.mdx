**设置 StackBlitz**
1. 访问 [astro.new](https://astro.new) 然后点击`Open in StackBlitz`按钮打开一个项目初始模板。

2. 点击右上角的`Sign in`以使用你的 GitHub 账号登录。

3. 点击 StackBlitz 编辑器窗口左上角的`fork`, fork此模板（保存到你的个人账号中）

4. 等待项目加载完毕，你将看到“项目初始模板”的实时预览。

**修改文件**

在文件窗口中，你应该可以看到 `src/pages/index.astro`。 点击打开，按照[你的第一行 Astro 代码](/zh-cn/tutorial/1-setup/3/)来修改此文件。

**创建 GitHub 仓库**

1. 点击文件列表顶部的 <kbd>Connect Repository</kbd> 按钮，为你的仓库重命名，然后单击 <kbd>Create repo & push</kbd>。 

2. 当你有修改要提交到 GitHub 时，点击工作区的左上角的提交`Submit`按钮将。点击此按钮输入提交信息`commit message`到你的仓库。

**部署你的网站**

如果你想部署到 Netlify，请跳至[将你的网站部署到网络](/zh-cn/tutorial/1-setup/5/)。

### 本单元目标

在本单元中，你将创建一个部署在 Github 且连接到 Netlify 的新项目。

在编写代码时，你会将修改定期提交到 GitHub。Netlify 将使用你的 GitHub 仓库的文件来构建你的网站，然后将其发布在互联网上的唯一地址，任何人都可以查看它。

每次你向 GitHub 提交更改时，都会向 Netlify 发送一条通知。然后，Netlify 将自动检测你的修改，重新部署并发布到实时站点。
