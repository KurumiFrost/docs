<Lede> 在本教程中，你将通过构建一个功能齐全的博客来学习 Astro 的主要功能，从零到完整部署！ 🚀 </Lede>

在这一过程中，你将：
- 建立你的开发环境 
- 为你的网站添加页面和博客文章
- 使用Astro组件构建
- 查询和处理本地文件
- 为你的网站添加交互
- 将你的网站部署到网络上

想预览一下最终效果吗？ 你可以在 [GitHub](https://github.com/withastro/blog-tutorial-demo) 或 [StackBlitz](https://stackblitz.com/github/withastro/blog-tutorial-demo/tree/complete?file=src%2Fpages%2Findex.astro)上查看最终效果。

:::note
如果你想用一个预先构建的 Astro 网站来探索 Astro， 你可以访问 https://astro.new 选择一个启动模板，然后在在线编辑器中打开编辑。
:::
